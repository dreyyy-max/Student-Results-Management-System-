# Student-Results-Management-System-
A Python console application built to automate student record management for a secondary school environment. The system allows teachers to add, update, and search student records stored in a dictionary data structure, with automatic grade calculation based on a defined grading scale.
# Python Management Systems

Two Python console applications built for managing school records and store inventory.

## Projects
- **Student Result Management System** — Manages student scores, calculates grades, and generates class statistics.
- **Inventory Management System** — Tracks product stock, flags low inventory, and generates store reports.
